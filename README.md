## What This Repository Is

This repository maps the complete intellectual lineage of the Human-AI 
Collaboration framework stack developed by Basil C. Puglisi from 2012 
through the present. It contains working papers, framework mappings, 
conceptual drafts, and in-progress documents that document how the stack 
was built, why each layer exists, and where the work is heading.

---

## The Framework Stack

| Layer | Framework | Origin | Function |
|---|---|---|---|
| 1 | Factics | Nov 2012 | Facts + Tactics + KPIs. Human thinking discipline before AI. |
| 2 | HAIA-RECCLIN | Sep 2025 | Seven AI roles. Structured single-platform governance. |
| 3 | HAIA-CAIPR | Mar 2026 | Cross AI Platform Review. Parallel multi-AI orchestration. |
| 4 | CBG | 2025 | Checkpoint-Based Governance. Constitutional human authority. |
| 5 | HEQ = AIS | 2025 | Human Enhancement Quotient. Measures human cognitive growth. |
| 6 | HAIA-Agent | 2026 | Automated orchestration layer. Zero cognitive work. |
| 7 | GOPEL | 2026 | Governance Orchestrator Policy Enforcement Layer. Federal infrastructure. |
| Branch | HAIA-CORE + HAIA-SMART | 2025 | Content evaluation and distribution. Activates when output is content. |
